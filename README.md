# 🔢 BitManipulation — LeetCode Solutions

A focused collection of **Bit Manipulation problems** solved in **Java**, sourced from LeetCode. This repository dives deep into bitwise operations — AND, OR, XOR, shifts — and how they can be leveraged to write highly efficient solutions for classic and modern competitive programming problems.

## 🔑 Topics & Techniques

- **XOR Tricks** — cancellation property, prefix XOR, parity detection
- **Bitwise AND / OR** — subarray optimization, range reduction, greedy maximization
- **Bitmask DP** — subset enumeration, assignment problems, minimum cost over states
- **Bit Counting** — Hamming distance, popcount tricks
- **Sliding Window + Bits** — maintaining bit frequency windows for subarray problems
- **Greedy on Bits** — bit-by-bit construction, reducing numbers to zero
- **Bit Hashing** — encoding substrings as bitmask keys for existence checks
